# 小小學習王 Little Learning Hero

免費、可離線使用的兒童學習遊戲，提供英文、數學、注音、國字等 20 多種練習玩法，適合 3–5 歲孩子以點選方式學習。

## 線上版本

- 正式站：https://bghut.com/kids/
- GitHub Pages：啟用後可由儲存庫的 Pages 網址開啟

## 特色

- 20+ 種中英雙語學習玩法
- 五分鐘每日學習旅程
- 寵物養成、金幣與獎章
- 錯題複習與學習紀錄
- PWA 支援，可加入手機桌面並離線使用
- 學習紀錄只保存在使用者自己的瀏覽器

## 使用方式

直接開啟 `index.html`，或以任何靜態網站伺服器部署本目錄。

> GitHub Pages 是靜態版本，因此伺服器排行榜與意見回報功能會自動退回本機模式；完整功能請使用正式站。
